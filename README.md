## Installing
clone the repo
```
git clone https://github.com/SharonV33/guess-that-album.git
```
install dependencies
```
npm install
```
start the program
```
npm run start
```
or start the project in developer mode (with nodemon)
```
npm run dev
```
navigate to localhost in the browser
```
http://localhost:8080/
```
